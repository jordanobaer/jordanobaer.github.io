Projects:

Elevator FSM

The objective of this project was to design and implement a finite state machine that controls an elevator of a building with 2 floors. The machine has enable as an input to determine whether the elevator should remain in the idle state or continue the sequence. It contains 3 outputs, one for the door, the floor and the movement of the elevator.

Elevator sequence:

  1. Open the door for 4 seconds
  2. Close the door
  3. Move the elevator to the destination floor in 4 seconds and display the current floor number using an LED
  4. Open the door for 4 seconds
  5. Close the door
  6. Go to Idle state
